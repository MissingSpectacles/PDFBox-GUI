# PDFBox GUI

## How to Use

Windows: "RUN-APPLICATION-(NATIVE).cmd".  
Non-windows: "RUN-APPLICATION-(JAR).cmd".  
or `java -jar "./Source Code/build/libs/PDFBoxGUI-0.0-SNAPSHOT.jar"`.
